## How to run

### Requires

1. create .env
2. add WEATHER_API_KEY
    - register [open_weather_map](https://openweathermap.org/)
3. add API_URL (run tests)(only on bun)

### bun (prefer)

```
bun install
```
```
bun run start
```

### npm 

```
npm install
```
```
npm run start
```
